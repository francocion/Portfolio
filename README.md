# Video Editor Portfolio (static)

A minimal static portfolio for a video editor using a simple black + beige theme.

What this includes
- `index.html` — site structure and sample content
- `styles.css` — black/beige theme, responsive layout, CSS animations
- `scripts.js` — reveal on scroll, modal video playback, small menu
- `assets/` — SVG thumbs and a `videos/README` with instructions

How to preview
1. Open `index.html` in a browser, OR run a local server:
   - Python 3: `python -m http.server 8000` then open `http://localhost:8000`
2. Replace thumbnails and video file paths in `index.html` with your own assets.

Customization quick tips
- Replace the placeholders (names, contact, project titles/descriptions).
- Replace `data-video-src` values for each `.card` with either a YouTube/Vimeo embed or local mp4.
- Tweak colors in `styles.css` under `:root` variables.
